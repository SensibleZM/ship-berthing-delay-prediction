# 基于高级特征工程与多模型融合的船舶靠泊延误风险预测研究
**机器学习 × 水路交通运输学科交叉研究**

## 项目背景
港口船舶靠泊延误是影响物流效率与运营成本的关键问题，传统预测模型存在“黑箱”特性，可信度与可解释性难以满足港口精细化管理需求。本项目通过机器学习方法实现延误风险预测，并分析关键影响因素。

## 项目简介
基于高级特征工程与多模型融合的船舶靠泊延误风险预测，属于机器学习×水路交通运输学科交叉研究，解决港口调度中延误预测精度与可解释性难以兼顾的问题。

## 研究框架
1.  数据预处理：多源船舶数据清洗、异常值处理、缺失值插补
2.  高级特征工程：业务驱动特征构造、时序特征编码
3.  模型训练：10种机器学习算法对比、5折分层交叉验证
4.  可解释性分析：SHAP框架解析核心影响因子（到港高峰时段、风速等）
5.  模型融合：F1分数幂次加权融合策略，提升泛化性能

## 核心内容
1.  **特征工程**：构造船舶类型-到港时段交互特征、对数变换特征等，提升模型对业务规律的捕捉能力；
2.  **模型训练**：对比10种机器学习模型，最优单模型（GBDT）F1=0.6928；
3.  **模型融合**：加权融合模型F1=0.6820，分析弱模型拖后腿的影响，验证融合池构成的重要性；
4.  **可解释性**：通过SHAP明确高峰时段、风速为关键影响因素，为港口调度提供可解释依据。

## 技术栈
Python / Pandas / LightGBM / XGBoost / CatBoost / SHAP / Scikit-learn

## 实验结果
| 模型                     | F1-Score |
|--------------------------|----------|
| GBDT（最优单模型）       | 0.6928   |
| CatBoost                 | 0.6916   |
| 加权融合模型（10基模型） | 0.6820   |

## 核心结果
- 最优单模型 GBDT（F1=0.6928）在本任务中展现出最强拟合能力；
- 加权融合模型（F1=0.6820）虽未超越单模型，但验证了融合池构成对性能的影响规律；
- SHAP分析明确了高峰时段、风速等关键影响因素，为港口调度提供了可解释的决策依据。
