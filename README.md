# 船舶靠泊延误风险预测研究
## 项目简介
基于高级特征工程与多模型融合的船舶靠泊延误风险预测，属于机器学习×水路交通运输学科交叉研究，解决港口调度中延误预测精度与可解释性难以兼顾的问题。

## 研究框架
1. 数据预处理：多源船舶数据清洗、异常值处理、缺失值插补
2. 高级特征工程：业务驱动特征构造、时序特征编码
3. 模型训练：10种机器学习算法对比、5折分层交叉验证、贝叶斯调参
4. 可解释性分析：SHAP框架解析核心影响因子（到港高峰时段、风速等）
5. 模型融合：F1分数幂次加权融合策略，提升泛化性能

## 核心成果
- 融合模型F1-Score：0.6812，较最优单模型提升12%
- 明确港口延误核心影响因子，验证业务逻辑有效性
